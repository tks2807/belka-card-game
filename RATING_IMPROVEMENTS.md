# Улучшения системы рейтинга в Белке

## Проблема старой системы

Предыдущая система рейтинга показывала только **процент побед (winrate)**, что было несправедливо по следующим причинам:

- 🎯 Игрок с 1 победой из 1 игры (100%) выше игрока с 50 победами из 60 игр (83%)
- 📊 Не учитывалось качество игры - очки, взятки, достижения
- 🎲 Новички могли легко попасть в топ, сыграв всего пару игр
- ⚖️ Отсутствовал минимальный порог для участия в рейтинге

## Новая справедливая система рейтинга

### 🎯 Два типа игроков

1. **Квалифицированные игроки** (5+ игр) - используют гибридный ELO рейтинг
2. **Новички** (менее 5 игр) - простой процент побед

### 🏅 Гибридный рейтинг (ELO + качество игры)

Объединенная система, которая учитывает **и силу противников, и качество игры**:

#### Принцип работы ELO:
- **Начальный рейтинг**: 1000 ELO для всех новых игроков
- **Победа над сильными**: Больше очков ELO (+15-40 за игру)
- **Победа над слабыми**: Меньше очков ELO (+5-25 за игру)
- **Поражение от слабых**: Большая потеря ELO (-15-40 за игру)
- **Поражение от сильных**: Малая потеря ELO (-5-25 за игру)

#### Расчет гибридного рейтинга:
```
1. Базовое изменение ELO = 32 × (Фактический результат - Ожидаемый результат)
2. Модификатор качества = очки × взятки × бонус_за_голую
3. Итоговое изменение = Базовое изменение × Модификатор качества (0.5-2.0)
```

#### Модификаторы качества игры:
- **Очки**: ±20% за результативность относительно личной средней
- **Взятки**: ±15% за тактическое мастерство
- **Голая победа**: +50% к изменению рейтинга
- **Общий модификатор**: от 0.5x до 2.0x

#### Командная игра:
- Учитывается средний ELO команды vs средний ELO команды противников
- Каждый игрок получает изменение ELO индивидуально с учетом своей игры
- ELO ведется отдельно для глобального рейтинга и для каждого чата

#### Примеры:
- 🔥 **Слабые (800) побеждают сильных (1200) с отличной игрой**: +52 ELO
- 😐 **Равные (1000 vs 1000) играют средне**: ±16 ELO  
- 🙄 **Сильные (1200) побеждают слабых (800) с плохой игрой**: +3 ELO
- 🏆 **Голая победа над равными**: +24 ELO вместо +16

### 📋 Новые команды

#### Рекомендуемые команды (с честным рейтингом):
- `/rating` - Справедливый рейтинг всех игроков
- `/ratingchat` - Справедливый рейтинг игроков этого чата

#### Старые команды (простой винрейт):
- `/leaderboardall` - Простая таблица по винрейту (все)
- `/leaderboardchat` - Простая таблица по винрейту (чат)


### ⚖️ Справедливость новой системы

1. **Минимальный порог**: Нужно сыграть 5 игр для участия в основном рейтинге
2. **Комплексная оценка**: Учитывается не только результат, но и качество игры
3. **Разделение новичков**: Новички соревнуются отдельно до получения опыта
4. **Бонусы за мастерство**: Дополнительные очки за сложные достижения

### 🔄 Миграция

- Старые команды `/leaderboardall` и `/leaderboardchat` продолжают работать
- Новые команды `/rating` и `/ratingchat` используют честную систему
- Данные автоматически пересчитываются для всех существующих игроков

### 🎮 Рекомендации игрокам

1. **Играйте больше**: Минимум 5 игр для попадания в основной рейтинг
2. **Играйте качественно**: Набирайте больше очков и взяток
3. **Стремитесь к достижениям**: Голые победы дают значительный бонус
4. **Используйте новые команды**: `/rating` вместо `/leaderboardall`

*Эта система обеспечивает справедливое соревнование между игроками разного уровня и поощряет качественную игру, а не только погоню за победами.*
